# Nvim-config
没有众多大佬所配置的nvim那样强大的扩展性，但配置通俗易懂(没有复杂的自定义函数，只需查看对应插件即可在此基础上自行配置)  

适用人群：想要一个较为完善的nvim，并在此基础上配置成独属于自己的nvim，但不熟悉lua高级语法的同志

## 安装
```bash
git clone https://github.com/LittleBear1025-xzh/nvim-config.git ~/.config/nvim
nvim +PackerSync
```
